Introduction

If you already have Bootstrap in your style sheet, skip this section.
To add Bootstrap CSS to your site, add the following line of code into the head section of your html file.
This makes use of a Content Delivery Network (CDN). 

      

      <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.1.3/css/bootstrap.min.css" integrity="sha384-MCw98/SFnGE8fJT3GXwEOngsV7Zt27NXFoaoApmYm81iuXoPkFOJwJ8ERdknLPMO" crossorigin="anonymous">

You can also download and use a local version of the CSS sheet. This is not recommended, but if you want to learn more here.



Containers

With bootstrap, you are able to create containers that will dynamically resize and can be divided into 1-12 sections. This is written as follows: 

      
      <div class="container">
          <div class="row">
    	       <div class="col-sm">
    	          The number of times this piece is repeated is the number of columns
    	         </div>
             </div>
           </div>

Buttons

With bootstrap, you are able to create buttons of many styles without the hastle of making them look nice. This is written as follows: 



      
      <button type="button" class="btn btn-XXXX">Title</button>


The different styles are displayed below with their names (what would replace XXXX in the code above).




Other button styles

Note: These can all be used with all above styles, primary is being used as an example.

Outline buttons 
      
       <button type="button" class="btn btn-outline-primary">Primary</button>
